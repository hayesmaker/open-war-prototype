# open-war-prototype
express prototype for the open-war-engine
 
[Features Wiki](https://github.com/hayesmaker/open-war-prototype/wiki/Features)

## install
`npm install`

## use as node module

```
var OWE = require('./module/index');

OWE.main();
```

## Run web client
- `npm start`
- `localhost:8000`
